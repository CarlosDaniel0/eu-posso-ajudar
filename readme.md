## Início Rápido
Para utilizar esse projeto você precisa ter instalado em sua máquina o [.NET Core CLI](https://dotnet.microsoft.com/download) e [Node.js](https://nodejs.org/en/).

### Frontend
```Javascript
cd frontend
yarn dev || npm run dev
```

### Backend
```C#
cd api
dotnet run
```
